HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element defines CSS styles on the webpage
  • The meta element defines metadata such as character set, discription, keywords, author and viewpoint
  • The script element embeds client-side scripts in the HTML docment
  • The link element connects an external resource to the HTML document
  • The title element displays the title of our page in the browser

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • A declaration is a rule that specifies which properties you want to style
  • A property is a way to add style to HTML such as color, font size, font type etc.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and swithces to it
  • git commit -m: commits branch with a message
  • git checkout file-name changes to the branch you want to work on
  • git pull fetches and merges commits from the remote branch to the local branch
  • git push transmits the local branch commits to the remote repo branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a scirpt.
  • A string is a sequence of text enclosed in quotes
  • A Boolean is a true/false value
  • an Array is a structure that allows you to store multiple values in a single reference
  • A Comment is a snippet of text that can be added with the code but the browser will ignore
